# Project Name & Description
Password Manager

NOTE: This project was created as part of a Udemy course ("100 Days of Code: The Complete Python Pro Bootcamp").

A Python-based password managing application in which users can input their email/username and password for any website/app, and the data will be securely stored locally and can be accessed anytime by calling the website/app name again. The app utilizes tkinter and json data.

## Features

* Displays a GUI for the user to input their email/username and password for a website/app.
* Stores the given data within a local, secure json file
* Includes randomly generated secure passwords if desired
* Displays login information, given the corresponding website/app name

## Technologies

* Python
* Tkinter
* Json File

## How It Works
* The GUI displays, with the user's most commonly used email/username already inputted (can be changed)
* The user inputs a website/app and a password (or chooses to have one randomly generated for them)
* The data is stored in a local, secure json file
* At any time, the user can run the app and enter a website/app and receive the correpsonding login information

## How To Run
* Edit EMAIL constant to be your most commonly used email/username
